# Automated Data Cleaning and Visualization Tool

## Overview

This project is a Python-based web application built using Streamlit that automates common data cleaning tasks and provides interactive visualizations for CSV datasets.

## Features

* Upload CSV datasets
* Remove duplicate rows
* Handle missing values
* Remove extra spaces
* Remove empty rows
* Display dataset overview
* Show statistical summary
* Generate interactive charts
* Download cleaned CSV files

## Technologies Used

* Python
* Pandas
* Streamlit
* Matplotlib
* Plotly

## How to Run

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## Project Structure

* app.py – Main application
* cleaning.py – Data cleaning functions
* charts.py – Visualization functions

## Output

The application cleans uploaded datasets, displays statistical information, generates visualizations, and allows users to download the cleaned data.
